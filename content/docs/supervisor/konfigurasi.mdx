# Konfigurasi Supervisor

Supervisor menggunakan file konfigurasi `.conf` untuk mendefinisikan proses yang akan dijalankan. Biasanya file-file ini diletakkan di folder `/etc/supervisor/conf.d/`

Setiap file bisa berisi satu atau beberapa program yang akan diawasi.

Setelah menambahkan konfigurasi baru, jalankan:
```bash
sudo supervisorctl reread
sudo supervisorctl update
```

## üìÅ Struktur Dasar Konfigurasi

```ini
[program:namaservice]
process_name=%(nama_program_isi_bebas)s
command=/path/ke/interpreter /path/ke/file/commandnya command
autostart=true
autorestart=true
user=nama_user
redirect_stderr=true
stderr_logfile=/var/log/nama_service.err.log
stdout_logfile=/var/log/nama_service.out.log
```

üîÅ Reload Supervisor
Setelah kamu buat atau ubah file `.conf`, jangan lupa reload:
```bash
sudo supervisorctl reread
sudo supervisorctl update
```
Untuk melihat status semua program:
```bash
sudo supervisorctl status
```
