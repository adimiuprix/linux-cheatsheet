# Installasi Supervisor

Berikut adalah langkah-langkah untuk menginstal Supervisor di berbagai sistem Linux yang umum digunakan. Supervisor bisa dijalankan di hampir semua distro, selama Python tersedia.

## üì¶ Installasi Cepat

<Tabs items={['Debian/Ubuntu', 'CentOS / RHEL / Fedora', 'Arch / Manjaro', 'Via Python (Cross-platform)']}>

<Tab value="Debian/Ubuntu">
```bash
sudo apt update
sudo apt install supervisor
```
Setelah terinstal, service Supervisor akan langsung aktif:
```bash
sudo systemctl status supervisor
```
</Tab>

<Tab value="CentOS / RHEL / Fedora">
```bash
sudo dnf install supervisor
```
Aktifkan dan jalankan Supervisor:
```bash
sudo systemctl enable supervisord
sudo systemctl start supervisord
```
</Tab>

<Tab value="Arch / Manjaro">
```bash
sudo pacman -S supervisor
```
Jalankan dengan:
```bash
sudo systemctl enable supervisord
sudo systemctl start supervisord
```
</Tab>

<Tab value="Via Python (Cross-platform)">
Cocok untuk Docker, VPS ringan, atau sistem yang tidak menggunakan package manager standar.
```bash
pip install supervisor
echo_supervisord_conf > supervisord.conf
```
Jalankan dengan:
```bash
supervisord -c supervisord.conf
```
</Tab>
</Tabs>

üìÅ Struktur File Konfigurasi
Setelah instalasi, Supervisor biasanya membaca konfigurasi dari:
```bash
/etc/supervisor/supervisord.conf

/etc/supervisor/conf.d/*.conf (untuk konfigurasi per aplikasi)
```
Contoh command untuk memuat ulang konfigurasi:

```bash
sudo supervisorctl reread
sudo supervisorctl update
```
<Callout type="warning"> 
Pastikan setiap file konfigurasi memiliki ekstensi `.conf` dan format yang valid, karena kesalahan kecil bisa menyebabkan semua service gagal dimuat.
</Callout>
