# Selamat Datang di Dokumentasi Supervisor

Supervisor adalah tool manajemen proses berbasis Python yang dirancang untuk memantau dan mengendalikan aplikasi di sistem operasi Unix/Linux. Cocok digunakan untuk menjalankan aplikasi sebagai background service (daemon) tanpa perlu menggunakan `systemd`.
