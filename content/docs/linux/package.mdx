## ğŸ¯ Update dan upgrade
Biasanya, ketika awal kita menjalanakan Linux atau Ubuntu, kita ingin update semua dependency atau packages yang terinstall, kita bisa menjalankan perintah berikut ini:

update dependency
```bash
sudo apt update
```
upgrade dependency
```bash
sudo apt upgrade
```
Jika kita ingin menjalankan kedua perintah di atas dalam satu perintah tunggal, tambahkan tanda `&&` di antara perintah.
```bash
sudo apt update && sudo apt upgrade
```
