# Pengenalan PM2

PM2 adalah **Process Manager** untuk aplikasi Node.js yang membantu kamu menjalankan, memonitor, dan mengatur aplikasi agar selalu berjalan. Dengan PM2, kamu bisa:

- Menjalankan aplikasi Node.js sebagai background service (daemon).
- Mengatur **auto-restart** jika aplikasi crash.
- Memantau performa aplikasi secara real-time.
- Menyimpan konfigurasi dan membuat aplikasi start otomatis saat server reboot.

PM2 sangat berguna untuk production environment karena membuat manajemen aplikasi menjadi lebih stabil dan mudah.
