# Installasi PM2

PM2 adalah process manager untuk Node.js yang membantu menjalankan aplikasi secara daemon, monitoring, dan auto-restart.

## Prasyarat

- Node.js sudah terinstall
- NPM atau Yarn tersedia

## Installasi via NPM

```bash
npm install pm2@latest -g
